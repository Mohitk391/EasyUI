Button

The Button component has basic 2 types, solid buttons and outlined buttons.

Solid Buttons

The Solid button components has basic 5 types, btn-primary, btn-round, btn-link, btn-hover, btn-disabled.

Outlined Buttons

The Outlined button components has basic 4 types, btn-outline-primary, btn-outline-round, btn-outline-icon, btn-outline-hover, btn-outline-disabled.

Icon and Floating Action Buttons

The floating action button can be implemented using btn-floating-action, and a button with an icon can be implemented using btn-icon.
